# tg-form-antdv

`tg-form-antdv` 是一个基于 [ant-design-vue](https://www.antdv.com/components/form-cn/) 与 [tg-validators](https://github.com/Tickly/tg-validators) 封装的表单组件

## 干什么的

简化了表单验证规则的编写，只需定义字段名称与验证器类型，无需手写错误提示。

## 安装

### npm
`npm i tg-form-antdv`
### yarn
`yarn add tg-form-antdv`
